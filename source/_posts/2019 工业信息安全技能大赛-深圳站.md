### Modbus 协议分析（Modbus 协议）

链接: https://pan.baidu.com/s/1RcC1VTPusecH9OqTvRPnPA?pwd=4wv3 提取码: 4wv3 

题目：黑客通过外网进入一家工厂的控制网络，之后对工控网络中的操作员站系统进行了攻击，最终通过工控协议破坏了正常的业务。我们得到了操作员站在攻击前后的网络流量数据包，我们需要分析流量中的蛛丝马迹，找到 FLAG

首先打开流量包，数据包都是关于 `Modbus/TCP` 的流量，下面的流量解析

MBAP Header（报文头，共 7 字节）

- Transaction ID（事务标识符，2 字节）：类似序号，请求和响应的 ID 必须一致

- Protocol ID（协议标识符，2 2字节）：固定为 `00 00`，代表 Modbus 协议

- Length（长度，2 字节）：表示后面还有多少个字节

- Unit ID（单元标识符，1 字节）：类似于从机地址（Slave ID）

PDU（协议数据单元）:

- Function Code（功能码，1 字节）： 它告诉设备要做什么（读还是写）

- Data（数据，可变长度）：具体读写的地址、数量或者具体的数值

我们重点关注功能码字段

| 功能码 (十进制) | 功能码 (十六进制) | 功能码名称 (英文) | 作用与含义 |

### 隐信道数据安全分析（MP3 Private Bit 隐写）

链接: https://pan.baidu.com/s/1q7SA4Yl3NyS5JFtKUnjtNw?pwd=92c4 提取码: 92c4

题目：安全分析人员截获间谍发出的秘密邮件，该邮件只有一个 mp3 文件，安全人员怀疑间谍通过某种 private 的方式将信息传递出去，尝试分析该文件，获取藏在文件中的数据？

MP3 音频文件是由无数个 "音频帧" 组成的。每个帧都有一个 4 字节的帧头（Header）

在帧头的结构中，第 23 个比特位（Bit 23）被称为 Private Bit（私有位）。这个位在官方标准中没有定义具体用途，完全由用户自定义

编写脚本，首先是定义对照表

比特率：

  - 索引 1 → 32 kbps

  - 索引 9 → 128 kbps

采样率：
  - 索引 0 → 44100 Hz（CD 音质）

  - 索引 1 → 48000 Hz（常见视频音轨）

```python
BITRATE_TABLE = [0, 32, 40, 48, 56, 64, 80, 96, 112, 128, 160, 192, 224, 256, 320, -1] # 比特率
SAMPLERATE_TABLE = [44100, 48000, 32000, -1] # 采样率
```

许多 MP3 文件在音频数据前附加了 ID3v2 标签（存储歌名、艺术家、封面等信息），我们需要识别跳过这个

结构：ID3v2 头部固定 10 字节：

- 第 0-2 字节：标识符 "ID3"（ASCII）

- 第 3-4 字节：版本号

- 第 5 字节：标志位

- 第 6-9 字节：标签总长度（不包含头部的 10 字节），采用同步安全整数编码（每个字节只使用低 7 位，最高位恒为 0）

```python
# 检查文件前 3 字节是否为 b'ID3'
# 若存在，读取第 6-9 字节，通过掩码 0x7F 取出每个字节的低 7 位，然后按大端序组合成一个 28 位整数
id3_size = (byte6 & 0x7F) << 21 |
           (byte7 & 0x7F) << 14 |
           (byte8 & 0x7F) << 7  |
           (byte9 & 0x7F)
```

因此，要完全跳过整个 ID3v2 标签，需要移动的字节总数 = 头部 10 字节 + 标签数据 size 字节

```python
pointer = 10 + id3_size
```

```python
import os
import sys

# 这两行是 MP3 帧头解析所需的对照表，用于将帧头中存储的索引值转换为实际的比特率和采样率
# 比特率：
  # 索引 1 → 32 kbps
  # 索引 9 → 128 kbps
# 采样率：
  # 索引 0 → 44100 Hz（CD 音质）
  # 索引 1 → 48000 Hz（常见视频音轨）
BITRATE_TABLE = [0, 32, 40, 48, 56, 64, 80, 96, 112, 128, 160, 192, 224, 256, 320, -1] # 比特率
SAMPLERATE_TABLE = [44100, 48000, 32000, -1] # 采样率

def bits_to_bytes(bit_str):
    """将 01 字符串每 8 位转换为 ASCII 字符"""
    byte_arr = bytearray()
    # 每 8 位一组，转换为一个字节
    for i in range(0, len(bit_str), 8):
        byte_bits = bit_str[i:i+8]
        if len(byte_bits) < 8:   # 不足 8 位的丢弃
            break
        byte_arr.append(int(byte_bits, 2))
    return byte_arr

def extract_private_bits(filename):
    """从 MP3 文件的每一帧中提取 private bit，拼接成二进制字符串"""
    with open(filename, 'rb') as f:
        data = f.read()
    
    pointer = 0
    file_size = len(data)
    
    # 1. 自动识别并跳过 ID3v2 头部标签
    # 许多 MP3 文件在音频数据前附加了 ID3v2 标签（存储歌名、艺术家、封面等信息）
    if data.startswith(b'ID3'):
        print("[+] 检测到 ID3v2 标签，正在计算大小并跳过...")
        # ID3v2 头部大小存储为 4 个 7 位字节（每个字节最高位为 0）
        id3_size = ((data[6] & 0x7F) << 21) | \
                   ((data[7] & 0x7F) << 14) | \
                   ((data[8] & 0x7F) << 7)  | \
                   (data[9] & 0x7F)
        pointer = 10 + id3_size   # 跳过 ID3 头部和标签数据
        print(f"[+] ID3v2 标签大小: {pointer} 字节")

    private_bits = ""
    frame_count = 0

    # 2. 开始精准按帧步进解析
    while pointer < file_size - 4:
        # 检查是否满足帧头条件：首字节 0xFF，次字节高 3 位为 111
        if data[pointer] == 0xFF and (data[pointer+1] & 0xE0) == 0xE0:
            byte1 = data[pointer+2]
            
            # 解析帧头中的关键字段
            bitrate_idx = (byte1 >> 4) & 0x0F      # 比特率索引
            samplerate_idx = (byte1 >> 2) & 0x03   # 采样率索引
            padding = (byte1 >> 1) & 0x01          # 填充位
            
            # 无效索引则跳过当前字节继续查找帧头
            if bitrate_idx == 0 or bitrate_idx == 15 or samplerate_idx == 3:
                pointer += 1
                continue
                
            # 查表得到实际的比特率和采样率
            bitrate = BITRATE_TABLE[bitrate_idx] * 1000
            samplerate = SAMPLERATE_TABLE[samplerate_idx]
            
            # 提取 Private Bit（帧头第 2 字节的最低位）
            private_bit = byte1 & 0x01
            private_bits += str(private_bit)   # 拼接所有帧的 private bit
            frame_count += 1
            
            # 计算当前帧的总长度：公式 144 * 比特率 / 采样率 + 填充字节
            frame_size = int(144 * bitrate / samplerate) + padding
            pointer += frame_size   # 跳过整个帧，继续解析下一帧
        else:
            pointer += 1   # 未找到帧头，前进 1 字节继续扫描

    print(f"[+] 扫描完毕，共成功解析 {frame_count} 个有效音频帧。")
    return private_bits

if __name__ == "__main__":
    target_file = sys.argv[1]
        
    raw_bits = extract_private_bits(target_file)
        
    print(f"\n[*] 提取到的原始二进制前64位: {raw_bits[:64]}...")

    flag_bytes = bits_to_bytes(raw_bits)
    
    print("\n" + "="*20 + " 尝试转码结果 " + "="*20)
    print(flag_bytes[:500].decode('utf-8', errors='replace'))
    print("="*54)
    
    with open("extracted_data.bin", "wb") as out_f:
        out_f.write(flag_bytes)
    print("[+] 完整二进制流已同步保存至本地: extracted_data.bin")
```
