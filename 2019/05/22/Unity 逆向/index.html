Unity

Unity 游戏的核心逻辑通常位于 C# 编译的 DLL（Mono）或 C++ 编译的 IL2CPP 文件中

编译方式

方式 脚本后端 包含文件 难度
Mono C# → IL → .NET DLL Assembly-CSharp.dll 等 低
IL2CPP C# → IL → C++ → 原生代码 libil2cpp.so / GameAssembly.dll + global-metadata.dat 较高
